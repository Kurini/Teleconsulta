Desenvolvido por Carlos Henrique Rorato Souza - Mestrando em Ciência da Computação
Universidade Federal de Goiás - 2020

Descrição Geral:


Pré-Requisitos:
	- Possuir o nodeJS instalado, com a biblioteca peer (npm install peer -g  e npm install peerjs -g);
	- Possuir recursos para chamadas de vídeo (webcam/microfone);

Instruções de Uso:
	- Para rodar o servidor de sinalização: na pasta "server", abra o WindowsPowerShell e digite o seguinte comando para iniciar:
		"peerjs --port 9000 --key peerjs --expire_timeout 60000 --alive_timeout 600000".
	- Se for necessário, em Index.html reorganize as informações sobre IP e portas.
	- Abra o arquivo index.html e siga as instruções de uso.
